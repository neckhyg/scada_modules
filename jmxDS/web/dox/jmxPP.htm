Overview

A reference to a JMX point consists of an Object name, a Attribute name, and optionally - if the attribute is a composite data type - a Composite item name. The simplest way to find values is to use the JMX server inspection feature.

The point Data type is the Mango representation of the value, not the JMX value type. Regardless, it is best to match the data type as closely with the JMX type for best results.
