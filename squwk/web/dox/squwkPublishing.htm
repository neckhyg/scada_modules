Overview

The Squwk.com publisher sends point updates (or change, as appropriate) to Squwk using the given access and secret key pair. Updates are sent in batches for maximum throughput.

Properties

The Access key and Secret key are needed to authenticate with Squwk. You can find it in the "Account" area when you log in to Squwk.com.

Published points

The Points section lists the points to be published. Point publishing can be enabled () or disabled (). Disabling a point allows it to not be used without having to explicitly delete it. The GUID value determines the point that will be updated.

To add a point to the point list, select it in the points list and click the icon. To delete an existing point, click its associated icon.
